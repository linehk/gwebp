gwebp 可以递归地将指定目录下的图片转换成同名的 webp 格式的图片。

## 安装

需要先行安装 webp，如：`brew install webp`，然后：

```bash
go install github.com/linehk/gwebp
```

## 使用

```bash
gwebp -r='.' -e='jpg' -a='-q 50 -lossless' -k=true
```

```bash
gwebp -r='.' -e='png' -a='-sns 70 -f 50 -size 60000' -k=false
```
